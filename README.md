# 医院管理系统 java295

## 项目概述

医院管理系统是基于SpringBoot和Vue的前后端分离系统，适用于医疗机构内部使用。该系统支持管理员、医生和患者三种角色，提供全面的医院信息管理功能。

## 技术栈

- **后端：**
  - SpringBoot
  - MyBatis-Plus
  - Redis
- **前端：**
  - Vue
  - Element-UI

## 系统角色

- **管理员：**
  - 管理医生、患者、挂号、药物、检查、病床、排班等信息
  - 进行数据统计分析，支持管理决策

- **医生：**
  - 管理挂号和住院申请
  - 查看和编辑个人信息
  - 提高工作效率

- **患者：**
  - 预约挂号
  - 查看挂号和住院记录
  - 管理个人信息，便于与医护人员沟通

## 功能模块

- **用户权限管理：**
  - 登录与注册
  - 权限控制

- **信息管理：**
  - 医生信息管理
  - 患者信息管理
  - 药物信息管理
  - 检查项目信息管理

- **业务处理：**
  - 挂号管理
  - 住院申请管理
  - 病床分配

- **数据统计：**
  - 挂号统计
  - 药物使用统计
  - 检查项目统计

## 推荐运行环境

- **JDK：** 1.8
- **数据库：** MySQL 8.0
- **项目管理工具：** Maven 3.8

## 核心亮点

- **全面的角色支持：** 涵盖管理员、医生和患者，满足不同用户的实际需求
- **前后端分离：** 基于SpringBoot和Vue，便于维护和扩展
- **权限控制：** 确保用户信息的安全和系统稳定运行
- **高效的后台管理：** 支持医院高效管理和决策
- **数据统计分析：** 提供多维度数据统计，支持医院精细化管理
